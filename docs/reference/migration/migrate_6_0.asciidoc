[[breaking-changes-6.0]]
== 6.0 重大改变

本节讨论将应用程序迁移至 Elasticsearch 6.0 需要注意的改变。

[float]
=== 6.0 之前创建的索引

Elasticsearch 6.0 可以读取 2.0 或更高版本创建的索引。如果存在 Elasticsearch 5.0 之前版本创建的索引，Elasticsearch 6.0 节点将不能启动。

[IMPORTANT]
.从 Elasticsearch 2.x 或之前版本重建索引
=========================================

为了 Elasticsearch 6.x 可读，需要将 Elasticsearch 2.x 或之前版本创建的索引重建为 Elasticsearch 5.x 版本的索引。

=========================================

[IMPORTANT]
.6.0 创建索引时将不支持多 mapping 类型
=========================================


一个索引拥有多 mapping 类型功能将从 6.0 移除。新索引将限制为单类型。这是移除 mapping 类型的第一步。 5.x 版本创建索引时仍将支持多 mapping 类型。

详细信息请参考 <<removal-of-types>> 。

=========================================
