[[supported-platforms]]
[float]
== 平台支持

官方支持的操作系统和 JVM 信息列表可从此获得：link:/support/matrix[Support Matrix]。
Elasticsearch 在列出的平台已通过测试，但在一些其他平台可能也能正常运行。

[float]
[[jvm-version]]
== Java (JVM) 版本

Elasticsearch 是基于 Java 构建的，正常运行至少需要 http://www.oracle.com/technetwork/java/javase/downloads/index.html[Java {jdk_major}]。
Elasticsearch 仅仅支持 Oracle 的 Java 和 OpenJDK，同时所有节点和客户端需使用同一 JVM 版本。

我们建议安装 *{jdk} 的 Java 版本或更高版本。我们建议使用 link:/support/matrix[supported]
http://www.oracle.com/technetwork/java/eol-135779.html[LTS version of Java]。
如果使用不支持的版本，Elasticsearch 将拒绝启动。

Elasticsearch 使用的 Java 版本可通过 `JAVA_HOME` 环境变量配置。
