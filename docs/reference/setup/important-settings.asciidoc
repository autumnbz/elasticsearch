[[important-settings]]
== Elasticsearch 重要配置

虽然 Elasticsearch 只需要很少的配置，但在投入生产之前还是有一些需要考虑的设置。

在投入生产环境之前，*必须* 考虑以下设置：

* <<path-settings,路径设置>>
* <<cluster.name,集群名>>
* <<node.name,节点名>>
* <<network.host,网络主机>>
* <<discovery-settings,发现设置>>
* <<heap-size,堆大小>>
* <<heap-dump-path,堆 dump 路径>>
* <<gc-logging,GC 日志>>
