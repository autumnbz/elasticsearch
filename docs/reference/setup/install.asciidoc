[[install-elasticsearch]]
== 安装 Elasticsearch

Elasticsearch 提供以下安装包格式：

[horizontal]
`zip`/`tar.gz`::

`zip` 和 `tar.gz` 包适合在任何操作系统上安装，同时也是大多数系统入门 Elasticsearch 的最简单选择。
+
<<zip-targz>> 或者 <<zip-windows>>

`deb`::

`deb` 包适用于 Debian、Ubuntu 和其他 Debian-based 系统。Debian 包可从 Elasticsearch 官网和 Debian 库下载。
+
<<deb>>

`rpm`::

`rpm` 包适用于 Red Hat、Centos、SLES、
OpenSuSE 和其他 RPM-based 系统。RPMs 可以从 Elasticsearch 官网和 RPM 库下载。
+
<<rpm>>

`msi`::

`msi` 包适用于最低安装了.NET 4.5 framework 的 Windows 64-bit 操作系统，同时也是 Windows
入门 Elasticsearch 的最简单选择。MSIs 同样可以从 Elasticsearch 官网下载。
+
<<windows>>

`docker`::

Docker 容器运行 Elasticsearch 的镜像可从 Elastic Docker 注册处下载。默认镜像预安装了
{xpack-ref}/index.html[X-Pack]。
+
{ref}/docker.html[Install {es} with Docker]
